# 📚 Daily Five Words (Collins 1)

这是一个自动化的英语词汇学习工具，旨在帮助用户通过 **间隔重复 (Spaced Repetition)** 和 **真实语境 (Contextual Learning)** 高效掌握英语词汇。该项目通过每日自动化抓取词库、生成带有深度辨析与真实新闻语境的卡片，并自动同步至 GitHub Pages 进行展示。

## 🌟 核心功能

* **智能词库抓取**：每日从 Collins 词库中自动抽取 5 个生词。
* **多维度解析**：
    * **深度释义**：整合有道词典与 DictionaryAPI，提供详尽的中英文解释。
    * **辨析讲解**：提供词汇的词性分析与多义项辨析。
    * **真实语境**：自动抓取 BBC、NYT 等国际主流媒体新闻，生成带有关键词高亮的真实例句。
    * **可视化学习**：自动调用 Wikipedia API 获取高清配图，辅助记忆。
* **交互式日历**：前端基于状态机重构，支持年月快速切换，查看历史学习记录。
* **云端同步**：支持通过移动端/PC端直接编辑备注，并一键提交修改同步至 GitHub 仓库。

## 🚀 工作流程

1.  **运行脚本**：程序自动从指定词库读取未学习的单词。
2.  **数据构建**：调用多个 API 获取释义、例句、搭配和图片。
3.  **HTML 渲染**：根据模板生成每日学习卡片。
4.  **枢纽更新**：自动编译 `index.html` 生成学习日历，方便按日期回顾。
5.  **GitHub 同步**：所有生成内容自动部署至 GitHub Pages。
